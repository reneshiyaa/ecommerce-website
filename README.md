## 📌 Project Description
This project is a responsive e-commerce website frontend developed using HTML, CSS, and JavaScript. It allows users to browse products, add items to cart, and view total price.

## 🚀 Features
- Responsive design
- Product listing page
- Add to cart functionality
- Cart page with total calculation
- Clean UI with Tailwind CSS

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Tailwind CSS
